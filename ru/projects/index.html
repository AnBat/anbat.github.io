Frontol Manager (в компании Атол)

Frontol Manager — специализированное программное обеспечение класса Middle-office, занимающее промежуточное положение между Front- и Back-офисом и предназначенное для оперативного управления объектами торговых предприятий. Frontol Manager позволяет централизованно управлять и администрировать кассовую линейку, а также оборудование в торговом зале (ВПЭ), контролировать деятельность и осуществлять транспорт данных между всеми узлами системы, а также выполнять и фиксировать операции, связанные с перемещением материальных ценностей предприятия. А главное, FM консолидирует информацию и автоматически производит ее двусторонний обмен в режиме синхронизации между кассами, back-офисом и оборудованием (весы).

QCurrencyEdit

Бибилиотека для управления валютами и деньгами в приложении. Состоит из нескольких классов, а именно:

  • QCurrency - класс для управления валютами
  • QExchangeRates - для для управления курсом обмена между валютами
  • QMoney - класс для управления деньгами
  • QMoneyEdit - ГУИ для управления деньгами (по типу spinbox-а)
